-Алгоритм и структура данных
воспользуемся отображеним из элемента списка в количество этих элементов в списке.
Так как порядок по условию не важен, воспользуемся неупордоченным отображением.
Так как изначальный размер списка не известен, воспользуемся динамическим массивом (д.м.) для сохранения порядка вхождения элементов списка
Ответ на задачу осуществляется с помощью итерирования по д.м., содержащего element исходонго списка, и проверки value мапа: если value = 1, то выводим key == element.

-Асимптотика
O(N), где N — размер входного списка.
